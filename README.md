# desafio-acc-back

A parte do BackEnd do desafio foi feito com Java/Spring bot, usando o banco de dados Postgres.

#1
-Editar o arquivo application.properties colocando os dados do banco [{nome_do_banco},{nome_usuario_banco},{senha_usuario_banco}]

#2
-Executar o comando `mvn clean install` para baixar as dependências do maven

#3
-Executar o comando `mvn spring-boot:run` para executar o projeto Spring boot que será startado na seguinte URL: "http://localhost:8080"

#!
Caso não tenha o ´mvn´ configurado pode usar a IDE intelliJ, ela fara o download das dependencias do projeto automaticamente e também é possível executar o projeto ao clicar no botão `RUN`
